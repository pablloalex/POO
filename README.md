# O que é um vetor em C++? Explique com suas palavras.
## é uma sequência de elementos do mesmo tipo, acessada por índices.

# Qual a diferença entre declarar um vetor com valor fixo e inicializá-lo com valores específicos?
## Valor fixo significa que o vetor é apenas declarado, mas seus elementos podem estar indefinidos até serem atribuídos valores.

## Inicialização é quando você fornece valores específicos para os elementos do vetor no momento da declaração.

# Explique o que significa índice e qual o intervalo válido para um vetor int notas[10];
## Índices são as posições dos elementos dentro do vetor e começam em 0.

## Para o vetor int notas[10];, o intervalo de índices válidos vai de 0 a 9.
